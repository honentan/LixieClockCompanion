# LixieClockCompanion
LixieClock with 0.96oled - show weather forecast and China lunar calendar.

拟辉光管时钟伴侣
升级辉光管时钟，增加0.96寸OLED，显示实时天气、明后天气、农历、工作日。

修改key_secret.h中的wifi信息和心知天气密钥，农历api密钥。
